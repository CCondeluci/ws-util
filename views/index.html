Reference Cards

One-Sheet JP Reference Cards with Translations

This utility retrieves a deck from EncoreDecks and collates all reference cards for that deck into one easily readable set of pages, rather than requiring a user to click and print all reference cards individually. The exact width and height of each reference card has been preserved, so the output from this utility should be acceptable for official WGP events. Reference cards will not bleed or break onto successive pages when printed. This will only work for decks that only contain JP cards.

To submit, simply copy and paste the deck code from any JP EncoreDecks URL into the box and click the button. Submission can take a while, as HotC does not like anyone hitting their endpoints quickly. Please be patient while the utility processes your deck.

Example: https://www.encoredecks.com/deck/ 9OxCN_EMf <-- Enter this code!
